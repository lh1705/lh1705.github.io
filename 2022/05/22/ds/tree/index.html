1 基本概念

  • 祖先。从根到当前结点的唯一路径的任意结点，成为该结点的祖先。
  • 子孙。当前结点是从根到当前结点唯一路径的任意结点的子孙。
  • 双亲。到当前结点的唯一路径上离当前结点最近的结点，成为该结点的双亲。
  • 孩子。当前结点是从根到当前结点的唯一路径上离当前结点最近的结点的孩子。
  • 兄弟。有相同双亲的结点称为兄弟。
  • 堂兄弟。处在同一层的结点互为堂兄弟。
  • 结点的度。一个结点的分支数（孩子数）。
  • 树的度。树中结点的最大度数。
  • 树的深度。树中结点的最大层数。
  • 分支结点（非终端结点）。度大于 0 的结点。
  • 叶子结点（终端结点）。度为 0 的结点。
  • 结点间的路径。两个结点之间所经过的结点序列。
  • 路径长度。路径上所经过的边的个数。
  • 树的路径长度。树根到每个结点的路径长度的综合。
  • 有序树。 树中结点的各子树从左到右是有次序的。
  • 无序树。 树中结点的各子树从左到右没有次序。

3.4 二叉排序树

  • 左子树上所有结点的关键字均小于根节点的关键字。
  • 右子树上的所有结点的关键字均大于根节点的关键字。
  • 左右子树又各是一个二叉排序树。

3.5 平衡二叉树

  • 树上任意结点的左子树和右子树的深度之差不超过 1。

5 树和森林

  • 森林的先根序列 == 对应二叉树的先序序列
  • 森林的中序序列 == 对应二叉树的中序序列
  • 含有 n 个结点，m 条边的森林，共有 n-m 棵树
  • 森林 F(T1, T2, T3) 的左子树结点数为 n(T1) - 1, 右子树结点数为 n(T2) + n(T3)
