3 Markdown

3.1 Basic config

Use Tagbar and markdown2ctags to show outline, vim-table-mode to format table, vim-markdwon to fold and lable jump and generate Toc.

3.2 Outline

To use markdown2ctags plug or self-define ctags's config.

Windows:

  • install plugin jszakmeister/markdown2ctags
  • add let g:tagbar_type_markdown into vimrc/ init.vim
  • complier markdown2ctags.py into .exe
  • set let g:tagbar_type_mardown's 'ctagsbin': /path/to/exe

Linux:

  • install plugin jszakmeister/markdown2ctags
  • add let g:tagbar_type_markdown into vimrc/ init.vim
  • set let g:tagbar_type_mardown's 'ctagsbin': /path/to/markdownctags.py

3.3 Toc

Use plug vim-markdwon to generate Toc.

  • :InsertToc [n]: Generate the unordered Toc which can use number to specify heading level.
  • :InsertNToc [n] Generate the nordered Toc which can use number to specify heading level.

4 Mapping

4.1 Alt mapping

Hit Ctrl-v (Ctrl-q in win) to go into visual-block mode, and hit the map-key, we will get some chars, use these chars to map the function. It likes: nmap <M-,> gt,and <M-,> is used as hitting-chars for Alt-,.
