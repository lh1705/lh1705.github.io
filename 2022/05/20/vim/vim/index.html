Vim 基础

lh1705

Vim

1 Basic

1.1 Insert

Key Description
i 插入到光标前
a 插入到光标后
s 插入到光标下
o 插入到下一行
O 插入到上一行
A 插入到行首
I 插入到行尾
S 插入到当前行

1.2 Move

Key Description
[n]w 向前移动到下一个单词开头
[n]W 向前移动到下一个单词开头，期间忽略特殊符号
[n]e 向前移动到下一个单词结尾
[n]E 向前移动到下一个单词结尾，期间忽略特殊符号
[n]b 向后移动到前一个单词开头
[n]B 向后移动到前一个单词开头，期间忽略特殊符号
[n]f[x] 向前移动到指定字符位置
[n]F[x] 向后移动到指定字符位置
[n]t[x] 向后移动到指定字符前一字符位置
[n]T[x] 向前移动到指定字符前一字符位置
0 移动到行首
^ 移动到当前行的第一个非空字符
$ 移动到行尾
- 移动到上一行开头
+ 移动到下一行结尾
nG 移动到指定行
n+ 向上移动指定行
n- 向下移动指定行
H 移动到当前屏幕顶部
M 移动到当前屏幕中间
L 移动到当前屏幕底部
[n]( 移动到句子开头
[n]) 移动到句子结尾
[n]{ 移动到段落开头
[n]} 移动到段落尾部
gg 移动到文档底部
G 移动到文档顶部
H 移动到屏幕顶部
M 移动到屏幕中间
L 移动到屏幕底部

1.3 Scroll

Key Description
[n]<C-f> 向下滚动一屏
[n]<C-b> 向上滚动一屏
[n]<C-d> 向上滚动半屏
[n]<C-u> 向下滚动半屏
[n]<C-e> 向下滚动一行
[n]<C-y> 向上滚动一行
zt/z<CR> 滚动到屏幕顶部
zz/z. 滚动到屏幕中央
zb/z- 滚动到屏幕底部

1.4 Delete

Key Description
dl(x) 删除当前字符
diw 删除当前单词
[n]dd 删除当前行
di_ 删除文本对象(单词、段落、句子) ([{
da_
d_ 删除到行首、行尾、文首、文末
[n]dt[c] 从当前字符向后删除到指定字符
[n]dT[c]
[n]df[c] delete form current char to specify char
[n]dF[c]

1.5 Select

Key Description
v/V 行选择
/ 块选择

1.6 Copy

Key Description
[n]yy 复制当前行
[n]Y 复制当前行
y_ 复制到行首、行尾、文首、文末
yi_ 复制文本对象（单词、句子、段落) （[{
ya_ 复制文本对象
[n]yt[c] 向前复制行内当前字符到指定字符前一字符间的内容
[n]yT[c] 向后复制行内当前字符到指定字符前一字符间的内容
[n]yf[c] 向前复制行内当前字符到指定字符间的内容
[n]yF[c] 向后复制行内当前字符到指定字符间的内容

1.7 Paste

Key Description
p

1.8 Search

Key Description
/ 从光标开始处向文件尾搜索
? 从光标开始处向文件首搜索
n 移动到上一个搜索结果
N 移动到下一个搜索结果

1.9 Replace

Key Description
r 单字符替换
R 进入替换模式，覆盖原内容
[range]s/find_str/replace_str/g 替换指定范围的字符

1.10 Folding

Key Description
zf 创建折叠标记
zd 删除折叠标记
zo 展开折叠
zc 折叠

Jump

Key Description
]] 跳到下一函数
[[ 跳到前一个函数
]} func end bracket
[{ func start bracket

1.10 Window

Key Description
CTRL-W h 移动到左窗口
CTRL-W j 移动到下窗口
CTRL-W k 移动到上窗口
CTRL-W l 移动到右窗口
CTRL-W t 移动到顶窗口
CTRL-W b 移动到底窗口
sp 水平分割窗口
vsp 垂直分割窗口
:resize +n 增加窗口高度
:resize -n 降低窗口高度

Useful trick

Key Description
0w 移动到行首第一个非空字符
gd 折行
J 多行合并
<C-l> 消除搜索高亮
YpVR_ Markdown Setext heading
source $MYVIMRC 重载 vimrc/init.vim 文件
12, 31 co 22 复制指定行到指定位置
12, 31 m 22 移动指定行到指定位置
