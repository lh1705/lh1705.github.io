快捷键

Key Description of function
F12 另存为
F4 重复上一步操作（如果可以）
F5 定位条件
Ctrl-k 插入超链接
Ctrl-Shift-F9 取消超链接
Ctrl-Shift-j 分散对齐
Ctrl-Enter 分页
Alt-MouseLeft 列选择
Alt-Shift-Left 提高标题等级
Alt-Shift-Right 降低标题等级
Shift-Delete 删除表格
Shift-Enter 软回车

常见问题

  • 样式列表中没有常见的样式

    被隐藏掉了（管理样式 - 推荐 - 选择隐藏的 - 显示）

  • 多级列表不能自动应用

    多级列表对应的格式有问题

常用技巧

  • 删除空行

    pp 替换成 ^p（要进行多次替换，如何还有，查找下一个，删除即可）

  • 奇数页设置水印

    先在编辑页眉、页脚中设置水印，然后再删除偶数页即可

常用域命令

Description Command
总页数 { NUMPAGES # "0" }
页眉标题

多级列表

  • 通过 alt - shift - left/right 列三行内容
  • 然后自定义多级列表
  • 最后应用即可

NOTE 自定义多级列表优势

  • 当指定标题格式时，标题会自动根据设定的多级列表继承前一标题格式。
  • 方便导出带有目录的 pdf 文件（在导出时，选项中勾选非打印信息）

图片

图片与题注同页

  • 选择题注 - 绘制文本框
  • 同时选中题注与图片，右键组合即可。

图片只显示一行

在段落设置中，将行距改为单倍行距。

表格

快速删除

选中 - shift-delete

表格不能重复标题行跨行

  • 选中表格，右键表格属性将允许跨页断行和标题重复出现取消打钩;
  • 然后选中表格中想要重复的标题行，右键表格属性将允许跨页断行和标题重复出现重新打钩，确定即可。
