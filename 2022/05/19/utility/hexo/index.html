5.2 常见问题

  • 报错 remote: Support for password authentication was removed on August 13, 2021

    • 从 2021 年 8 月 13 日起，GitHub 不再支持密码认证，改为使用个人访问令牌凭证
      （Personal access tokens)，
    • 获取 token: Settings -> Developer settings -> Personal access tokens -> Generate new
      token

  • deploy 更新 token 后，仍旧报错

    • 为该仓库全局设置 token。repo: https://token@github.com/user/repo

  • 网页打不开 404

    • GitHub Pages 只支持 public 仓库
    • 未指定 GitHub Pages source
    • GitHub Pages 生成网页需要时间（1-2分钟）

  • 网页未更新

    • 本地数据提交失败
    • GitHub Pages 生网页需要等待 1-2 分钟

6 评论系统

cactus 主题内置两种评论系统 disqus 和 utterances。

disqus 是个成熟的评论系统，并且支持匿名评论。但 disqus 评论需要登录外网账号，虽然可以匿名评论但每次评论需要填写额外信息，还需要验证，特别麻烦。

utterances 是个轻量级的评论系统，评论只需登录 GitHub 账号即可，其评论系统借助GitHub issus 来进行管理，支持 markdown 语法，界面好看。缺点只能进行一级评论，不能进行二级评论（回复）。

那个，这两个我都用过，disqus 缺点太明显，没用多久就放弃了。utterance 什么都好，就好只能进行一级评论，这实在太遗憾。不过今天突然发现了一个宝藏 giscus。

giscus 评论系统可以看做 utterance 的升级版，一方面它弥补了 utterance 的缺点，其支持回复（二级评论），另一方面评论系统借助 GitHub discussion 进行管理，对评论管理更加高效合理。

ok，让我们开始部署。
