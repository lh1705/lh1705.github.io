GitHub 部署

部署流程

  1. GitHub 创建名为 user_name.github.io 的 public repo
  2. 安装 hexo-deployer-git 以及更新 _config.yml
  3. 部署

常见问题

  • 报错 remote: Support for password authentication was removed on August 13, 2021

    • 从 2021 年 8 月 13 日起，GitHub 不再支持密码认证，改为使用个人访问令牌凭证
      （Personal access tokens)，
    • 获取 token: Settings -> Developer settings -> Personal access tokens -> Generate new
      token

  • deploy 更新 token 后，仍旧报错

    • 为该仓库全局设置 token。repo: https://token@github.com/user/repo

  • 网页打不开 404

    • GitHub Pages 只支持 public 仓库
    • 未指定 GitHub Pages source
    • GitHub Pages 生成网页需要时间（1-2分钟）

  • 网页未更新

    • 本地数据提交失败
    • GitHub Pages 生网页需要等待 1-2 分钟
