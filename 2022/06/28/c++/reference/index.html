引用的本质是一个指针常量

Note int * const p 表示 p 是一个指针常量，即指针的指向不可改变，只能 改变其所指向对象的值。而引用初始化后，不能更改引用的原因就在于此。

常量引用

Note 常量引用是指用 const 关键字来修饰引用。

NOTE 不能把 const 对象、字面值或者需要类型转换的对象传递给普通的引用形参。 因为引用实质上绑定一个对象，其作为对象的别名，但因为引用实现实际上是顶层 cosnt（指针是个常量），引用形参初始化会忽略 const，即忽略对象的 const 属性， 那绑定的就不是原对象了。
