封装

意义

  • 将属性和行为作为一个整体，表现生活中的事物
  • 将属性和行为加以权限控制

基本术语

  • 属性 -> 成员属性 | 成员变量
  • 行为 -> 成员函数 | 成员方法
  • 实例化 -> 根据类模板创建一个对象的过程

访问权限

keyword 类内 类外
public 可访问 可访问
private 可访问 不可访问
protected 可访问 不可访问

strcut 与 class 区别

strcut 的成员权限默认为 public， class 成员的权限默认为 private
