片段编写

extends

引用合并其他 snippet 文件

priority

指定片段匹配的优先级，priority 默认为 0。如果之前片段的 priority < 0，则新建片段会覆盖之前发布的片段。

片段定义

trigger_word

trigger_word 可以包含空格。

options

  • options 控制片段的行为。
  • options 由单字符表示。
  • options 字符可以自由组合但不包含空格。
options description
b 设置只能在行首的非空字符处触发。
i 设置可在单词内（行内）触发。
W 设置只能在前后为空字符时触发。会覆盖 i 选项。
r 设置 trigger_word 可为 python 正则表达式。
t 片段定义中包含 tab，展开时忽略 vim 配置的 tab 格式。
s 如果片段编辑行前包含空白字符，且未编辑，那么在跳转到下一个编辑位置时，清除该行前的空白字符
m 消除片段定义中行内右侧的空白字符。 hh hh_ _ -> hh hh
e 自定义片段触发条件。
A 自动触发片段。
