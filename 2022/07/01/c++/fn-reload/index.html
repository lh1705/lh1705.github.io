重载条件

  1. 函数名相同
  2. 函数在同一个作用域下
  3. 函数形参列表的形参数据类型不同 / 个数不同 / 顺序不同

NOTE 函数的返回值不能作为函数重载的条件。

顶层 const 不可重载

实参初始化形式形参时忽略顶层 const，即在匹配函数时忽略顶层 const 进行匹配，进 而导致带顶层 const 和无 const 的函数是同一函数，造成函数二义性调用，导致不能 重载。

NOTE const int a 与 int const a 是等价的，都是限定指针指向的对象是常量。 非常量引用也是顶层 const，不能进行重载。

常量引用、底层 const 作为重载条件

非常量引用就是顶层 const，即限定指针是一个常量，同样不能作为重载条件。底层 const 限定指针指向的对象是一个常量，const 不忽略，可以作为重载条件。但常量引 用可以作为重载条件。
