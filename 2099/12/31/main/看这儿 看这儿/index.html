这个博客怎么用

好吧，我承认我想不到好的词，就姑且这样说。

  • 所有带下划线的文字都是可以点击跳转的。
  • 大部分绿色的文字也是可点击的。
  • 有些奇奇怪怪的图标也是可以点击的。
  • 你觉得可以点的，那大概率就是可以点的。

OK，完了。就这么简单，其他有趣的东西你要自己试着点啊。

这个博客为什么长这个样

喏，你现在看到的，网站的所有元素，例如，这文字为什么只有这么大，这英文字体是什么字体，为什么评论在最下面等等一些奇奇怪怪的问题，这些都是这个网站主题所决定的。

回到问题上来，为什么这个博客长这个样。当然，是由我决定的嘞。我自己选了个叫做 cactus 的主题模板，然后根据需求对其进行了小修小改，然后就变成这样了。

怎么样，是不是很好看。

=== 文章

对于博客里面的这些文章，额，怎么说呢，如果你想看的话，还是只看日记比较好。

为什么？因为除了日记以外的其他文章，连我自己都不想看...

你可能会问，文章就这么点吗。不是的，实在是许多东西拿不出手或者写的太烂。至于你所 能看见的这些，只是我放上去充数的...

=== Topics

这里呢，主要是对文章的这些分类作以解释。其实这部分内容还没完全想好，不过在这先提一提，之后有时间了再慢慢完善。

你现在在 Topics 所能看见的这些分类，在这儿，我们姑且用版块来称呼。对于这些版块，你或许有一定的了解，或许见名知意，知道这个版块是干嘛的。所以呢，我在这里就主要介绍一些特殊的版块。

Vim ===

Vim 代指的是 vim 文本编辑器。主要收录自己的 vim 学习笔记。

Utility ===

Ulitity 是实用工具的意思。顾名思义，这里收录的是一些特别实用的工具，包括但不限于 CLI 工具、特殊的 GUI 效率工具、标记语言等。
